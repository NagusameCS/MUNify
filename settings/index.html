Manual Instructions

Deploying an Apps Script Web App
  1. Create a new Google Apps Script project
  2. Paste contents of apps_script/Code.js & appsscript.json
  3. Deploy > New Deployment > Web App (execute as: Me, access: Anyone with link)
  4. Copy the Web App URL into the field above
  5. Click Register with Server
